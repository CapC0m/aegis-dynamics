# Stoke Engine

A Kerbal Space Program mod adding a differential-throttle second-stage engine inspired by Stoke Space's Andromeda.

## Features

- Ring of thrust chambers around an actively-cooled heatshield
- Differential-throttle TVC (no gimbal)
- Modular chamber count (3-24, live in VAB)
- Five diameter variants: 1.25m, 1.875m, 2.5m, 3.75m, 5m
- Waterfall plume effects with atmospheric response
- Heatshield-grade thermal tolerance
- MechJeb compatible

## Dependencies

- Module Manager
- B9PartSwitch
- Waterfall
- Stock Waterfall Effects (recommended)

## Installation

Copy the contents of `GameData/` into your KSP `GameData/` folder.
